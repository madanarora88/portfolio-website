# PlateRater Design Images

Add your PlateRater design JPEG files to this folder. Name them:
- 1.jpg
- 2.jpg
- 3.jpg

(Add more as needed: 4.jpg, 5.jpg, etc.)

Then update the `images` array in `src/data/profile.ts` (inside the PlateRater side project) to match your file names, e.g.:
["/assets/plate-rater/1.jpg", "/assets/plate-rater/2.jpg", "/assets/plate-rater/3.jpg"]
